# Share Network — Python Expedition (Volunteer Course)

This repository contains **3 beginner assignments** and a **final project (Snake Game)** for the Share Network Python Expedition class.

- Instructor: **Ahmed**
- Audience: Beginners (first steps in programming)
- Submission options:
  1) **GitHub Fork + Pull Request** (preferred)
  2) **Google Drive upload** (allowed)

## How to use this repository
1. Open the assignment folder.
2. Read the **README.md** instructions.
3. Start from the **starter** code.
4. Save your solution in your own submission folder.

---

## Submission (Option A — GitHub: Fork + Pull Request)
1. Fork this repo to your GitHub account.
2. Create a new branch:
   - `submission-yourname-a01` (for Assignment 01)
3. Add your solution here:
   - `student-submissions/<your-name>/assignment-01/fizzbuzz.py`
4. Commit and push.
5. Open a Pull Request.

Repeat for A02, A03, and the final project.

---

## Submission (Option B — Google Drive)
If you cannot use GitHub, submit via Google Drive:
1. Create a folder named: `Python Expedition - <Your Name>`
2. Inside, create subfolders:
   - `Assignment-01`, `Assignment-02`, `Assignment-03`, `Final-Project`
3. Upload your `.py` files and any required screenshots.
4. Share the folder link with the instructor (view access is enough).
