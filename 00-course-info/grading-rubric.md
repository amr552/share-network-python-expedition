# Grading Rubric (Simple)

Each assignment is graded on:
- Correctness (50%)
- Code clarity (20%)
- Naming & structure (15%)
- Edge cases (10%)
- Comments / readability (5%)

Final project (Snake Game):
- Runs without errors (30%)
- Core gameplay works (move, food, score, game over) (40%)
- Code structure (functions/classes) (15%)
- UX polish (controls, restart, instructions) (10%)
- README + screenshots (5%)
